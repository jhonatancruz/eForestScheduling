Please ensure that your spreadsheet file above follows the following constraints.
The program may not function properly if these constraints are in any way violated.

  1. The CLA Schedule must be the first sheet from left, within the spreadsheet file.
  2. The first 3 rows are for the Title and other information. Use them as you wish.
  3. Put Column Headers on Row 4.
  4. Class Listings must start on Row 5.
  5. The last row of the spreadsheet is determined by the first row for which the First Column (“Term”) has no value.
  6. At the very least, the following columns must be present, and in exactly the specified order:
    (These titles were taken verbatim from the Spring '18 schedule and are not representative of what the titles can be.)
    1. Term
    2. Fac Dept
    3. Subj
    4. #
    5. Sec
    6. XL
    7. Course Title
    8. Instructor
    9. Adjunct Y/N (for whom?)
    10. Overload Y/N (for whom?)
    11. Actual Credits
    12. Days
    13. Times
    14. Course Cap
    15. Final Exam Y/N
    16. Classroom Needs
    Notes:
    • Column Names can be modified as you like, but the order of columns from left-to-right MUST be preserved.
    • Any number of additional columns can be added, and will have no effect on the running of the program.
  7. The Days column must contain the days that the classes meet in the usual shorthand that is used. Look at the following examples:
    • For a class that meets on Tuesdays and Thursdays, please enter only the characters TR in this field.
    • For a class that meets Mondays, Thursdays and Fridays, please enter only the characters MRF.
    • For TBD classes, entering TBD is accepted. Note that room allocations will not be processed for classes marked as TBD.
  8. The Times column must contain one time slot only, in either of the following two formats ... hh:mm-hh:mm (OR) hh.mm-hh.mm . Please ensure that the hh and mm are separated by either a colon (:) or a period (.) only and not any other character. Also make sure that the two times are separated only by one hyphen (-) and no other character. Please make sure no other characters, such as class meeting days, etc, are present in the field.
    • Listing multiple time slots for classes that meet at different times during the week is currently not supported.
    • For TBD classes, entering TBD is accepted. Note that room allocations will not be processed for classes marked as TBD.
